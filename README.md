# Sudoku-Game
A Java Swing-based Sudoku game with GUI, input validation, and automatic puzzle solving using the backtracking algorithm.

# Features

- Interactive 9x9 Sudoku board
- User input validation
- Solve Sudoku puzzles automatically
- Reset the board

# Technologies Used

- Java
- Java Swing
- Object-Oriented Programming
- Backtracking Algorithm
- MVC Architecture
